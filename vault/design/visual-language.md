# Visual language

Our visual style is restrained, calm, and editorial. We're not a tech startup; we're a working tool people use every day.

## Type

- Sans: Inter Tight 400/500/600
- Serif: Newsreader for long-form prose
- Mono: JetBrains Mono for code

## Colors

- Ink `#1a1a1a` for primary text
- Paper `#fbfaf6` for canvas
- Muted `#73726c` for secondary
- Accent `#185fa5` for links and active states

See [[design/components]] for how these compose.
