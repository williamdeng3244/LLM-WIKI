# UI components

A short list of patterns we use. Refer back to [[design/visual-language]] for tokens.

## Buttons

- Primary: ink background, white text, used for the main action only
- Secondary: white with thin black border, used for everything else
- Heights: 28px in dense UI, 32px in main flows, 36px in modals

## Modals

Centered overlay, paper background, max-width 720px for forms, 1100px for review queues. See [[product/personal-agents]] for an example flow.

## Badges

Lowercase, 10px, letter-spaced. Used for stability levels, revision status, and tags.
